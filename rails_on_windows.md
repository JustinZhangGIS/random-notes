# rails on windows

## 安装--railsinstaller

参考[教你10分钟内在Windows上完成Rails开发环境的安装和配置](http://www.cnblogs.com/tambor/archive/2011/12/25/rails_anzhuang_railsinstaller.html)。

建议先卸载掉原有安装的Ruby.

## 编辑器

## Hello World

在cmd下，先使用

	rails new

创建一个工程，然后：

	ruby server

运行服务。使用`http://localhost:3000/`进行访问。
